
Using Python 3.6.3.

You still have to install these libraries for Python3.
 - numpy
 - matplotlib
 - csv
 - arrow


Enter the root folder, type in

python main.py

in terminal or just click "Run" in IDE.


Datasets are in the datasets forlders. Data should be organized like the csv files in this folder. (The "timestamp" and the "close" are the two most important columns.) About 30 records in a file is preferred.
